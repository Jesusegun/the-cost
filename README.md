# The Cost

A consequence engine that makes distraction time emotionally expensive.

## Philosophy

This is a mirror, not a planner.
- No morning commitments
- No notifications
- No streaks
- Visceral confrontation with time wasted

## The Five Pillars

- **Faith** - Prayer, stillness, alignment
- **Wealth** - Deep work, skill building, income
- **Impact** - Initiatives, leadership, service
- **Growth** - Reading, learning, clarity
- **Relationships** - Presence, listening, connection

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- localStorage (no backend)
- PWA-ready

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
